安装
开始使用 Milvus node.js 客户端的推荐方法是使用 npm（Node 软件包管理器）在项目中安装依赖项。
npm install @zilliz/milvus2-sdk-node
# or ...
yarn add @zilliz/milvus2-sdk-node
这将下载 Milvus node.js sdk，并在 package.json 文件中添加依赖项。
