监控
Milvus CDC 通过 Grafana 面板提供全面的监控功能，使您能够可视化关键指标，并确保您的变更数据捕获 (CDC) 任务和服务器健康的顺利操作。
CDC 任务的指标
要开始使用，请将
CDC-grafana.json
文件导入 Grafana。这将添加一个专门用于监控 CDC 任务状态的仪表盘。
CDC Grafana 仪表板概述
：
Milvus-cdc-dashboard
关键指标解释：
任务
：处于不同状态（包括
初始
、
运行
和
暂停）
的 CDC 任务数量。
请求总数
：Milvus-CDC 收到的请求总数。
请求成功
：Milvus-CDC 收到的成功请求数。
任务数
：一段时间内处于
初始
、
暂停
和
运行
状态的任务数。
任务状态
：单个任务的状态。
请求数
：成功请求数和总请求数
请求延迟
：通过 p99 请求的延迟、平均值和其他统计数据。
复制数据速率
：读/写操作的复制数据速率
复制时滞
：读/写操作的
复制时滞
：读/写操作的复制时滞。
api 执行次数
：Milvus CDC 不同 API 的执行次数。
center ts
：读/写任务的时间戳。
