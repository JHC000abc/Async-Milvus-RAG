安装 Milvus Go SDK
本主题介绍如何为 Milvus 安装 Milvus Go SDK。
当前版本的 Milvus 支持 Python、Node.js、GO 和 Java SDK。
要求
需要使用 GO（1.17 或更高版本）。
安装 Milvus GO SDK
通过
go get
安装 Milvus GO SDK 和依赖项。
$ go get -u github.com/milvus-io/milvus/client/v2
